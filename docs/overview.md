# Overview

**Xyraiq™** is an open, ISO-style benchmark that stress-tests Large Language Models for:
1. Long-session memory retention  
2. Logical consistency under prompt fatigue  
3. Ethical & spiritual reasoning under contradictory cues  

The benchmark outputs a deterministic **Stress Signature™** (hash) and badge (Bronze → Platinum).
