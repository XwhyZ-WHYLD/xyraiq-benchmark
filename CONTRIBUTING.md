## Quick start
1. **Fork** the repo and create a feature branch:<br>`git checkout -b feat/my-amazing-idea`
2. **Install dev deps**: `pip install -e .[dev]`
3. Add or update **tests** in `tests/`.
4. Ensure `pytest` passes and `pre-commit run --all-files` is clean.
5. Push to your fork and open a **Pull Request** that links to an open Issue.

## Commit style
Follow [Conventional Commits](https://www.conventionalcommits.org/)  
Examples: `feat: add tone-drift detector`, `fix: handle empty prompt pack`.
